A Toy Redis clone built following the ["Build Your Own Redis" Challenge](https://codecrafters.io/challenges/redis).

Redis commands supported:

- [x] `PING`
- [x] `ECHO`
- [x] `TYPE`
- [x]  Key-Value: `SET`, `GET`
- [X]  List: `RPUSH`, `LPUSH`, `LRANGE`, `LLEN`, `LPOP`, `BLPOP`
- [ ]  Stream:
   - [ ] `XADD`
   - [ ] `XRANGE`
   - [ ] `XREAD`
- [ ] Transactions:
   - [ ] `INCR`
   - [ ] `MULTI`
   - [ ] `EXEC`
   - [ ] `DISCARD`
